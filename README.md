Реализована работа следующих генераторов псевдослучайных чисел:
-	линейный конгруэнтный генератор (LCG);
-	генератор на основе алгоритма Блюм – Блюма – Шуба (BBS);
-	линейный рекуррентный генератор (LFSR)
